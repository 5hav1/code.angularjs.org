$modelOptionsProvider

  1. - $modelOptions
  2. - provider in module ng

Here, you can change the default settings from which ngModelOptions directives inherit.

See the ngModelOptions directive for a list of the available options.

Properties

  • defaultOptions

    The default options to fall back on when there are no more ngModelOption directives as ancestors. Use this property to specify the defaultOptions for the application as a whole.

    The initial default options are:

    • updateOn: default
    • debounce: 0
    • allowInvalid: undefined
    • getterSetter: undefined
    • timezone: 'undefined'
